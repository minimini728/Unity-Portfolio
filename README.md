# 꿈의 왕국: 영원한 보금자리
# 게임 개요
5가지의 행동을 통해 미션을 수행해 나가며 폐허가 된 꿈의 왕국을 되살리는 게임

● 장르: 수집형 캐주얼 시뮬레이션
● 플랫폼: Android
● 개발 엔진: Unity
● 개발 기간: 3개월
● 형상 관리: SVN
● 개발 인원: 프로그래머 5인
